# خانه ایرانی - تم وردپرس

## توضیحات
تم خانه ایرانی یک تم حرفه‌ای برای وردپرس است که با ووکامرس سازگار بوده و برای فروشگاه‌های آنلاین طراحی شده است.

## ویژگی‌ها
- ✅ سازگار با ووکامرس
- ✅ طراحی ریسپانسیو
- ✅ پشتیبانی از Elementor
- ✅ Customizer پیشرفته
- ✅ استایل‌های مدرن و زیبا
- ✅ پشتیبانی از فونت فارسی Vazir
- ✅ بهینه‌سازی برای SEO

## نصب و راه‌اندازی

### 1. نصب تم
1. فایل تم را در پوشه `wp-content/themes/` آپلود کنید
2. از طریق پنل مدیریت وردپرس، تم را فعال کنید

### 2. فعال‌سازی ووکامرس
1. افزونه WooCommerce را نصب و فعال کنید
2. تنظیمات اولیه ووکامرس را انجام دهید

### 3. تنظیمات Customizer
1. به Appearance > Customize بروید
2. بخش‌های مختلف تم را سفارشی کنید:
   - Hero Section
   - رنگ‌ها
   - تایپوگرافی
   - چیدمان
   - فوتر
   - شبکه‌های اجتماعی

## ساختار فایل‌ها

```
khane-irani/
├── inc/
│   └── customizer.php          # تنظیمات Customizer
├── woocommerce/                # قالب‌های ووکامرس
│   ├── archive-product.php     # صفحه لیست محصولات
│   ├── single-product.php      # صفحه تک محصول
│   ├── content-product.php     # نمایش محصول در حلقه
│   ├── cart/                   # صفحه سبد خرید
│   ├── checkout/               # صفحه پرداخت
│   └── global/                 # فایل‌های عمومی
├── functions.php               # توابع اصلی تم
├── woocommerce.css            # استایل‌های ووکامرس
├── style.css                  # استایل اصلی
└── README.md                  # این فایل
```

## سفارشی‌سازی

### تغییر رنگ‌ها
از طریق Customizer > Colors می‌توانید رنگ‌های اصلی تم را تغییر دهید.

### تغییر فونت
در Customizer > Typography می‌توانید فونت‌های متن و عناوین را تغییر دهید.

### تنظیمات Hero Section
در Customizer > Hero Section می‌توانید:
- عنوان اصلی
- توضیحات
- متن دکمه
- لینک دکمه

را تنظیم کنید.

## پشتیبانی از ووکامرس

این تم به طور کامل با ووکامرس سازگار است و شامل:

- صفحه لیست محصولات
- صفحه تک محصول
- سبد خرید
- صفحه پرداخت
- صفحه حساب کاربری
- صفحه سفارشات

## بهینه‌سازی

### عملکرد
- استفاده از CSS Grid برای چیدمان
- انیمیشن‌های بهینه شده
- تصاویر ریسپانسیو

### SEO
- ساختار HTML5 استاندارد
- Schema markup برای محصولات
- بهینه‌سازی برای موتورهای جستجو
