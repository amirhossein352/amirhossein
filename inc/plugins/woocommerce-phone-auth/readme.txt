افزونه مدیریت و فروش اکانت‌های هوش مصنوعی با قابلیت ورود و ثبت نام با شماره تماس

== Description ==

افزونه WooCommerce AI Account Manager امکان فروش و مدیریت اکانت‌های هوش مصنوعی (مانند ChatGPT) را فراهم می‌کند. این افزونه همچنین امکان ورود و ثبت نام کاربران با استفاده از شماره تماس و کد یکبار مصرف را دارد.

**ویژگی‌های اصلی:**

* فروش و مدیریت اکانت‌های هوش مصنوعی
* تقسیم اکانت‌ها بین چندین کاربر
* مدیریت خودکار تخصیص اکانت‌ها
* تعیین مدت اعتبار اکانت‌ها
* ورود و ثبت نام با شماره تماس
* ارسال کد تایید از طریق پیامک
* اتصال به پنل فراز اس ام اس
* شورت کد برای نمایش در هدر
* طراحی ریسپانسیو و زیبا
* پشتیبانی از تم تاریک
* استایل‌دهی پنل کاربری ووکامرس

**نحوه استفاده:**

1. افزونه را فعال کنید
2. از منوی تنظیمات > ورود با شماره تماس، کلید API فراز اس ام اس را وارد کنید
3. محصولات AI Account را ایجاد کنید و اکانت‌ها را اضافه کنید
4. شورت کد `[wcpa_header_auth]` را در هدر قرار دهید
5. از منوی WooCommerce > AI Accounts برای مدیریت اکانت‌ها استفاده کنید

== Installation ==

1. فایل افزونه را در پوشه `/wp-content/plugins/woocommerce-phone-auth/` آپلود کنید
2. افزونه را از طریق منوی 'افزونه‌ها' در وردپرس فعال کنید
3. از منوی 'تنظیمات' > 'ورود با شماره تماس' تنظیمات را انجام دهید
4. شورت کد `[wcpa_header_auth]` را در هدر قرار دهید

== Frequently Asked Questions ==

= آیا نیاز به پنل فراز اس ام اس دارم؟ =

بله، برای ارسال پیامک نیاز به پنل فراز اس ام اس دارید.

= چگونه کلید API را دریافت کنم؟ =

از پنل فراز اس ام اس خود کلید API را دریافت کنید.

= آیا افزونه با سایر افزونه‌ها سازگار است؟ =

بله، افزونه با سایر افزونه‌های ووکامرس سازگار است.
